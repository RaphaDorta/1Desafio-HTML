O que é HTML

HTML significa HyperText Markup Language (Linguagem de Marcação de Hipertexto, em português). É a linguagem padrão utilizada para criar e estruturar páginas da web.

HTML fornece uma maneira de estruturar o conteúdo de uma página da web, como texto, imagens, vídeos e links, utilizando elementos e tags. Cada elemento HTML é representado por uma tag, que é delimitada por colchetes angulares, como tag. As tags são usadas para envolver e definir diferentes partes do conteúdo, indicando ao navegador como exibi-las.

Além disso, HTML pode ser combinado com CSS (Cascading Style Sheets) para adicionar estilo e formatação ao conteúdo, como cores, fontes e layout. E também pode ser complementado com JavaScript para adicionar interatividade e funcionalidades dinâmicas às páginas da web.

No geral, HTML é a espinha dorsal da web, fornecendo a estrutura básica para a criação de conteúdo e páginas da web acessíveis e interativas.

O que são tags

Tags HTML são elementos usados para estruturar e formatar o conteúdo em uma página da web. Elas são elementos fundamentais da linguagem HTML (Hypertext Markup Language). Cada tag HTML é representada por um nome entre sinais de menor que (<) e maior que (>), e pode ter atributos que fornecem informações adicionais sobre a tag.

¨

As tags HTML são usadas para criar diferentes tipos de elementos em uma página da web, como texto, imagens, links, formulários, listas e muito mais. Cada tag tem uma função específica e determina como o conteúdo dentro dela será interpretado e exibido pelo navegador.

Por exemplo, a tag "p" é usada para criar parágrafos, a tag "img" é usada para inserir imagens, a tag "a" é usada para criar links, e assim por diante.

As tags HTML são essenciais para a estruturação semântica de uma página da web, ajudando os navegadores a entenderem a hierarquia e o significado do conteúdo, além de permitir o estilo e a formatação adequados por meio de CSS (Cascading Style Sheets) e a interatividade por meio de JavaScript.

Paragáfos

Em HTML, a tag p é usada para criar parágrafos. Essa tag é usada para envolver blocos de texto que devem ser tratados como um único parágrafo. Quando o navegador encontra a tag p, ele insere uma quebra de linha antes e depois do texto dentro da tag, criando assim um espaço vertical entre os parágrafos.

As tags p são amplamente utilizadas para estruturar o conteúdo de uma página web, tornando mais fácil para os desenvolvedores e os navegadores entenderem e exibirem o texto de maneira organizada e legível.

Listas ordenadas

"OL" em HTML é uma abreviação de "Ordered List" (Lista Ordenada). É uma das tags usadas para criar listas em HTML. Quando você usa a tag "ol", está criando uma lista na qual os itens são enumerados sequencialmente.

cada item da lista é definido pela tag li (ListItem), e a lista inteira é envolvida pela tag ol. Quando essa lista é renderizada em um navegador, os itens serão exibidos em ordem numérica.

Você pode usar o atributo type na tag "ol" para especificar o tipo de numeração (por exemplo, números, letras, algarismos romanos) e o atributo start para especificar o número inicial da enumeração.

Listas Desordenadas

A tag "ul" em HTML é uma abreviação de "Unordered List" (Lista Não Ordenada). Ela é usada para criar listas nas quais os itens não são numerados sequencialmente, mas geralmente são marcados com bullets (pontos, círculos, quadrados, etc.).

cada item da lista é definido pela tag "li" (ListItem), e a lista inteira é envolvida pela tag "ul". Quando essa lista é renderizada em um navegador, os itens serão exibidos sem uma ordem numérica, mas geralmente com marcadores como pontos, círculos ou quadrados.

A tag ul é frequentemente usada quando a ordem dos itens não é importante, mas é necessário agrupá-los em uma lista.

Ancoração

A tag "a" em HTML é usada para criar hiperlinks, ou seja, elementos que permitem aos usuários navegar de uma página para outra ou dentro da mesma página. Ela é uma das tags mais importantes para a construção da estrutura de navegação em páginas da web.

a tag a envolve o texto "Link para exemplo" e o atributo href especifica o URL para o qual o link aponta. Quando um usuário clica neste link, será redirecionado para a URL especificada.

Além disso, você também pode usar a tag a para criar links internos dentro da mesma página, referenciando elementos com IDs específicos.

o atributo href contém o ID do elemento para o qual o link deve levar. Quando clicado, o navegador rolará automaticamente para a seção da página que possui o ID correspondente.

É importante lembrar que os links devem conter um destino válido no atributo href para funcionarem corretamente.

Edição de texto

As tags "strong" e "i" são exemplos de tags em HTML usadas para alterar a apresentação do texto. Aqui está uma explicação sobre cada uma delas:

  • A tag strong é usada para enfatizar o texto, geralmente tornando-o visualmente mais destacado ou mais importante. Ela indica que o texto contido dentro dela deve ser exibido em negrito. No entanto, seu significado semântico é de importância, não apenas de estilo. É recomendado usar "strong" quando o texto deve ser semanticamente mais importante, como títulos, palavras-chave importantes ou avisos críticos.

  • A tag "i" é usada para colocar ênfase em texto, geralmente alterando sua aparência para itálico. No entanto, seu significado semântico é de ênfase, não apenas de estilo. É recomendado usar "i" quando o texto deve ser exibido em itálico por razões estilísticas ou para indicar um termo técnico, uma expressão estrangeira ou um pensamento alternativo.

  • A tag "em" é usada para colocar ênfase em texto, geralmente alterando sua aparência para itálico. Assim como a tag "i", ela é usada para indicar ênfase ou destaque de texto, mas com um significado semântico de ênfase.

  • A tag "b" é usada para destacar texto, geralmente tornando-o visualmente mais escuro ou em negrito. No entanto, seu uso não implica um significado semântico, ao contrário da tag "strong". A tag "b" é usada principalmente para fins de estilo.

  • A tag "u" é usada para sublinhar texto. No entanto, o uso excessivo de sublinhado pode prejudicar a legibilidade, portanto, é geralmente desencorajado em favor de outros métodos de destaque.
